Patch notes [1/28/23]

  • Added controls and info on main
  • Centered Game screen

    • Controls

  • [WASD] - [W] moves up, [A] moves left, [S] moves down, [D] moves right

    • Gameplay

  • [Goal] - Simply walk around the screen
  • [Matching Slime Ball] - Change color to match the slime ball to gain a point
  • [Non-Matching Slime Ball] - If you collect a slime that is a different color you lose a point
